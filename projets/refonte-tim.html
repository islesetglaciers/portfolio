Refonte du site du TIM
Automne 2020

Projet d'équipe réalisé dans le cadre du cours de Production Web. Le but du projet était de faire la refonte du site officiel du programme, du design à la programmation. Le site n'est malheureusement pas accessible mais le code est disponible sur GitHub et un vidéo du site est aussi disponible plus bas.

  • Langages de programmation et outils utilisés :
    # SASS
    # JS
    # PHP
    # MySQL
    # Figma
  • Rôles et tâches :
    # Conception et implémentation de la base de données (MySQL)
    # Code de la section cours (HTML, SASS, PHP)
    # Co-chef d'équipe
    # Structure HTML/SASS de base du site
    # Design des interfaces
